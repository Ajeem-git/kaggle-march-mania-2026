# Kaggle - March Machine Learning Mania 2026

This repository contains my optimized machine learning pipeline and feature engineering approach for the **March Machine Learning Mania 2026** competition on Kaggle.

### Technical Overview
- Implemented robust Cross-Validation (`GroupKFold` by season) to prevent data leakage.
- Enhanced feature engineering using score-based differentials and corrected Massey rankings.
- Built a multi-seed ensembling architecture and scaled Logistic Regression with optimized blend weights.
- Features modular scripts for data loading, feature engineering, and model training.

*(Note: Datasets and generated submission CSVs are ignored in this version control to keep the repository lightweight).*
